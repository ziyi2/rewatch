# rewatch

## Start

run the webpack

```javascript
npm run dev
```
>Tip: you should install pm2 to manager the app's process!

run the App

```javacript
npm start
```



stop the App

```javacript
npm stop
```

## React Flux

```javascript
npm install flux --save
```

>Tip: see Example `client/javascript/flux-src`


## React Router

```javascript
npm install react-router --save
```

>Tip: see Example `client/javascript/router-src`
