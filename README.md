# rewatch

## Start

启动方式

```javascript
npm run dev     //webpack
npm start       //node app
```


Make sure your server is running then visit [http://localhost:3000](http://localhost:3000)

>Tip: you should install `pm2` to manager the App's process, install `webpack-dev-server` to watch the files.


stop the App

```javacript
npm stop
```
